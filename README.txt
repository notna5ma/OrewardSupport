# Oreward Support Site

This is a simple static support website for App Store Connect.

Files:
- index.html = Support URL page
- privacy.html = Privacy Policy URL page
- style.css = styling

Before publishing:
1. Replace every `YOUR_EMAIL_HERE` with your real support email.
2. Edit the Privacy Policy if your app uses accounts, ads, analytics, online leaderboards, cloud saves, or collects any user data.
3. Upload the files to GitHub Pages, Netlify, Vercel, or any static web host.

GitHub Pages quick setup:
1. Go to github.com and create a new public repository named `oreward-support`.
2. Upload these files into the repo root.
3. Go to Settings > Pages.
4. Under Build and deployment, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / root
5. Save.
6. Wait 1-3 minutes.
7. Your Support URL will be:
   https://YOUR_GITHUB_USERNAME.github.io/oreward-support/
8. Your Privacy Policy URL will be:
   https://YOUR_GITHUB_USERNAME.github.io/oreward-support/privacy.html

App Store Connect:
- Support URL: paste the index page URL.
- Marketing URL: leave blank.
- Privacy Policy URL: paste the privacy page URL.
